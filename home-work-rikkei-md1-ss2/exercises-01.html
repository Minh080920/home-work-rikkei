Hướng dẫn cài đặt phần mềm VScode

  1. Tải bộ cài đặt từ trang chủ chính thúc
  2. Mở file vừa tải và ấn "NEXT"để tiến hành cài đặt
  3. Hoàn tất và tiến hành khởi động phầm mềm để sử dụng

Giải thích thuật ngữ của web

HTML
Bộ khung của cấu trúc trang web, định hình nền tảng và cấu trúc thông tin.
CSS
Định hình phòng cách, bố cục và trải nghiệm thị giác cho giao diện trang web
JS
Vận hành logic, điều khiển tương tác của trang web và xử lý dữ liệu linh hoat
